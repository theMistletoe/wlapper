# Usage
<!-- usage -->
```sh-session
$ npm install -g wlapper
$ wlapper COMMAND
running command...
$ wlapper (-v|--version|version)
wlapper/1.0.2 darwin-x64 node-v16.13.0
$ wlapper --help [COMMAND]
USAGE
  $ wlapper COMMAND
...
```
<!-- usagestop -->
```sh-session
$ npm install -g wlapper
or
$ npx wlapper

$ wlapper (-v|--version|version)
wlapper/1.0.0 darwin-x64 node-v14.18.1
$ wlapper --help [COMMAND]

USAGE
  $ npx wlapper [docker image name]
...
```
